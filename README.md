    <article class="markdown-body entry-content" itemprop="text"><h1><a href="#lc2ditp" aria-hidden="true" class="anchor" id="user-content-lc2ditp"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>LC2Ditp</h1>
<h2><a href="#bivariate-polynomial-interpolation-on-lissajous-chebyshev-points" aria-hidden="true" class="anchor" id="user-content-bivariate-polynomial-interpolation-on-lissajous-chebyshev-points"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Bivariate polynomial interpolation on Lissajous-Chebyshev points</h2>
<p>            <a href="/WolfgangErb/LC2Ditp/blob/master/img/2Dpicture1.jpg" target="_blank"><img src="/WolfgangErb/LC2Ditp/raw/master/img/2Dpicture1.jpg" width="370" style="max-width:100%;"></a>             <a href="/WolfgangErb/LC2Ditp/blob/master/img/2Dpicture2.jpg" target="_blank"><img src="/WolfgangErb/LC2Ditp/raw/master/img/2Dpicture2.jpg" width="370" style="max-width:100%;"></a></p>
<p>Version: 0.3 (01.10.2016)</p>
<p>Written by <a href="http://www.math.uni-luebeck.de/mitarbeiter/dencker/index.php" rel="nofollow"> Peter Dencker</a> and <a href="http://math.hawaii.edu/~erb/index.html" rel="nofollow"> Wolfgang Erb</a></p>
<h2><a href="#description" aria-hidden="true" class="anchor" id="user-content-description"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Description</h2>
<p>The package LC2Ditp contains a Matlab implementation for bivariate polynomial interpolation on
general Lissajous-Chebyshev points. This package syntesizes various interpolation schemes known in the literature.</p>
<p><br><br></p>
<p>            <a href="/WolfgangErb/LC2Ditp/blob/master/img/LC44.jpg" target="_blank"><img src="/WolfgangErb/LC2Ditp/raw/master/img/LC44.jpg" width="370" style="max-width:100%;"></a>             <a href="/WolfgangErb/LC2Ditp/blob/master/img/Gamma44.jpg" target="_blank"><img src="/WolfgangErb/LC2Ditp/raw/master/img/Gamma44.jpg" width="370" style="max-width:100%;"></a></p>
<p><br><br></p>
<p>The general description of the Lissajous-Chebyshev nodes and the polynomial interpolation is provided in the article [1].</p>
<ul>
<li>
<p>To <strong>test</strong> the package use <strong>main_example.m</strong></p>
</li>
<li>
<p>To <strong>plot</strong> the node points and the spectra use <strong>plot_LC2D.m</strong></p>
</li>
</ul>
<p><br><br></p>
<p>            <a href="/WolfgangErb/LC2Ditp/blob/master/img/LC54.jpg" target="_blank"><img src="/WolfgangErb/LC2Ditp/raw/master/img/LC54.jpg" width="370" style="max-width:100%;"></a>             <a href="/WolfgangErb/LC2Ditp/blob/master/img/Gamma54.jpg" target="_blank"><img src="/WolfgangErb/LC2Ditp/raw/master/img/Gamma54.jpg" width="370" style="max-width:100%;"></a></p>
<p><br><br></p>
<p>As special cases it contains the following interpolation schemes:</p>
<ul>
<li>The Padua points for the frequency parameters (m+1,m) and (m,m+1) [2,3].</li>
<li>The Morrow-Patterson-Xu (MPX) points for the frequency parameters (m,m) [9].</li>
<li>Interpolation schemes based on single degenerate Lissajous curves for
the frequency parameter (m,m+p), where m and p are relatively prime [5].</li>
<li>Interpolation schemes based on single non-degenerate Lissajous curves for
the frequency parameter (2m,2m+2p). [6]</li>
</ul>
